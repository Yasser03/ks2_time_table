# Kumon-style Multiplication Timetable App

## Overview
This application is designed to help users learn and practice their multiplication tables in a fun and interactive manner, inspired by the Kumon method.

## Features
- Interactive UI for selecting multiplication tables.
- Timed quizzes to test multiplication speed and accuracy.
- Progress tracking to see improvement over time.
- Engaging graphics and sounds to enhance the learning experience.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Yasser03/ks2_time_table.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ks2_time_table
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
- To start the application, run:
  ```bash
  npm start
  ```
- Follow the on-screen instructions to begin practicing multiplication.
